| Boundary | Description |
|----------|-------------|
| hyperparameter -> Loss constructor | User-supplied q / delta / alpha cross into der formulas; out-of-domain values (q<1, delta<=0, alpha outside [0,1]) can produce NaN/Inf that poisons training |
| der formula -> leaf/score reduction | A NaN der1/der2 propagates into the histogram/leaf sums and silently corrupts the model |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-06.1.01-01 | Tampering (DoS) | Lq der2 `|target-approx|^(q-2)` for q<2 (diverges near residual zero) | mitigate | Fixture pins q>=2 (Newton-clean); range-guard q>=1 in the Loss constructor -> typed CbError; q<2 routes to Gradient leaf (no der2 use), never panic (Pitfall 6) |

| T-06.1.01-03 | Tampering | NaN/Inf der leaking into leaf reduction | accept | compare_stage NaN-rejects (compare.rs:55-58) so any NaN der surfaces as an oracle failure, not a silent bad model; no external input surface (in-process numeric lib) |
