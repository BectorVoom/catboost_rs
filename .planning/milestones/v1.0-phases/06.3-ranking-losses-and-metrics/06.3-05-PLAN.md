gradient). This plan widens `EvalMetric::eval` with group structure (D-6.3-05), transcribes each

`libs/metrics/`, and gates all nine ≤1e-5 against frozen catboost 1.2.10 fixtures. It depends only
on the grouped fixture corpus (Plan 01) — independent of the loss waves, so it runs in parallel with
Wave A (Plan 02).

Purpose: Close LOSS-05 / SC-2 — the deterministic, Python-reachable ranking metrics.

per-metric ≤1e-5 oracle + frozen fixtures.
