  <behavior>
    - feature_weights: multiplicative on candidate gain — a split on float feature f scales `gain *= feature_weights[f]` (default 1.0 → no-op). Same insertion site as `cat_feature_weight`.
    - first_feature_use_penalties: subtractive — `score -= first_feature_use_penalties[f] * penalties_coefficient` only when feature f is not yet used anywhere in the model being built.
    - per_object_feature_penalties: subtractive — `score -= per_object_feature_penalties[f] * penalties_coefficient * unused_doc_count` where unused_doc_count is the whole-fold doc count when feature f is globally unused (symmetric path), else 0.
    - All-default params (empty vecs / coefficient default) → candidate scores byte-identical to the pre-6.6 path.
  </behavior>
  <action>
    Add four `BoostParams` fields with the doc-comment + paired `*_default()` convention (boosting.rs:88-340): `feature_weights: Vec<f64>` (default empty → treated as 1.0 per feature), `first_feature_use_penalties: Vec<f64>` (default empty → 0.0), `per_object_feature_penalties: Vec<f64>` (default empty → 0.0), `penalties_coefficient: f64` (`penalties_coefficient_default() -> 1.0`, upstream default). Cite upstream defaults in each doc comment.

    In tree.rs add `feature_weight(feature: usize, feature_weights: &[f64]) -> f64` returning `feature_weights.get(feature).copied().unwrap_or(1.0)` (sibling of `cat_feature_weight`, tree.rs:1095). Multiply it into the candidate gain at the SAME site `cat_feature_weight` is applied (the float-feature analogue of `GetSplitFeatureWeight`, greedy_tensor_search.cpp:980-988).

Track per-feature "already used" across the boosting model state. All sums (the per-object unused-doc count, any aggregation) route through `cb_core::sum_f64` (D-08); index access via `.get(...)` (no panic, `indexing_slicing` deny). NO embedded `#[cfg(test)]` — production file only.

    The default (empty-vec) path MUST leave `candidate.score` numerically untouched — gate each penalty branch on the param being non-empty, mirroring the `has_ctr` discriminator discipline (D-6.6-05).
