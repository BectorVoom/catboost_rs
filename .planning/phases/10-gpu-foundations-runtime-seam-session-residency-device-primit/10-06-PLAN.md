<objective>
Replace the plain one-u32-per-cell cindex with the upstream bit-packed grouped `WriteCompressedIndex` layout addressed by `TCFeature{Offset,Mask,Shift,OneHot}` (GPUT-15) — the single input every later histogram kernel consumes. This is the HIGH-ripple change: the bin-extraction expression changes in every consumer from a direct load to `(word>>Shift)&Mask`. Route all consumers through ONE `read_bin` accessor and oracle the packed layout bit-exact BEFORE wiring it into the tree.

Purpose: Memory efficiency is a first-class constraint and every histogram kernel's address arithmetic must be right once (D-07). Borders stay host (CPU quantization is the ≤1e-5 reference).
Output: gpu_runtime/cindex.rs (packer + accessor + oracle); migrated pointwise_hist + partition read.

- Migrated bin-extraction in `kernels.rs` pointwise histogram + `gpu_runtime/mod.rs` partition read
