<objective>
W5: Minimal Variance Sampling (MVS) on device (GPUT-17) — CatBoost's DEFAULT GPU sampler, distinct from GPUT-09's Poisson/Bayesian/Bernoulli. MVS is inherently a device reduction over resident derivatives: per-block (`BlockSize = 8192`) optimal threshold over `sqrt(λ + der²)` candidates + inverse-probability reweight. The device reproduces `cb-train/src/bootstrap.rs::mvs_sample_weights`/`calculate_threshold`/`single_probability`: `p = single_probability(|der|, threshold) = |der|>threshold ? 1 : |der|/threshold`; `weight = 1/p` w.p. `p` else `0` via `NextUniformF`; `lambda = GetLambda(...)`.

Purpose: cover the default GPU sampler on device, reusing the Plan-05 block/segmented sort and the Plan-06 device RNG discipline.
Output: `mvs_device` kernel + self-oracle + MVS gate arm.
</objective>
