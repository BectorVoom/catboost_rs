- **After every plan wave:** the wave's per-loss oracle test **plus** the full scalar oracle suite (D-04 must stay green through every wave).
- **Before `/gsd-verify-work`:** full per-stage oracle (borders/splits/leaf-values/staged N-dim approx/predictions) green for every named loss.
- **Max feedback latency:** per-crate test runtime (seconds).
