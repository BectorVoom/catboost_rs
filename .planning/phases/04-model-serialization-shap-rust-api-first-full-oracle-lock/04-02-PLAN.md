<objective>
Build the first real train->predict slice: the pure-Rust CPU apply path (MODEL-02), the prediction-type transforms (LOSS-06), and the remaining binary-classification losses CrossEntropy + Focal (LOSS-01). After this plan a user can train a binclf/regression model and get oracle-correct predictions in every in-scope prediction type — the apply path runs with no GPU toolchain present.

Output: `cb-model::apply` (CPU tree-evaluation), `cb-model::predict` (PredictionType transforms), CrossEntropy + Focal in `cb-compute`/`cb-backend`, and oracle tests for apply, prediction types, and the two new losses.
</objective>
