## Trust Boundaries

| external .cbm file -> load_cbm | PRIMARY untrusted-input surface: malformed/malicious magic, size field, FlatBuffers buffer |
| external model.json -> load_json | Untrusted JSON: deeply nested / huge / malformed arrays |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-03-01 | Denial of Service | .cbm size field -> huge allocation / OOB read | mitigate | Bound declared ui32 core size against actual file length before slicing/allocating; checked `.get(range)`; verifying `root_as_*` (upstream caps depth=64) — typed Deserialize error on overrun |
| T-04-03-02 | Tampering | truncated/corrupt FlatBuffers buffer | mitigate | flatbuffers crate verifier rejects; map to `ModelError::Deserialize`; no `_unchecked` accessor |
| T-04-03-03 | Tampering | wrong magic / FormatVersion | mitigate | Reject non-`CBM1` magic and non-`FlabuffersModel_v1` FormatVersion with typed `SchemaVersion`/`Deserialize` error |
| T-04-03-04 | Denial of Service | huge/deeply-nested model.json | mitigate | serde_json is safe by default; no unsafe; indexing_slicing/unwrap denied so malformed arrays return typed errors, never panic |
| T-04-03-05 | Tampering | integer overflow on offsets/sizes | mitigate | `u32::try_from` / checked `usize` conversions throughout (indexing_slicing deny enforces) |
</threat_model>
