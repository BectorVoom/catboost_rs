- **After every task commit:** Run `cargo test -p cb-backend --no-default-features --features rocm kernels::gradient_gpu -- --nocapture` (the touched der oracle).
- **After every plan wave:** `cargo test -p cb-backend --no-default-features --features rocm kernels::gradient_gpu` + `cargo build -p cb-backend --no-default-features --features wgpu` + `cargo build -p cb-backend --no-default-features --features cuda` (compile-gate the non-rocm backends).
- **Before `/gsd-verify-work`:** rocm der oracle green on gfx1100; wgpu builds+runs on host; cuda compiles; `cb-compute`/`cb-core`/`cb-model` byte-unchanged.
- **Max feedback latency:** ~90 seconds (per-crate, disk-pressure constraint forbids full-workspace test links).
