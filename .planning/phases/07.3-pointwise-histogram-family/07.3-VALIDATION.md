## Sampling Rate

- **After every task commit:** Run the touched family's quick run (e.g. `cargo test -p cb-backend --features rocm kernels::pointwise_hist::nonbinary_8bit -- --nocapture`)
- **After every plan wave:** Run all histogram families on rocm + `wgpu` build+run on host + `cuda` compile-only
- **Before `/gsd-verify-work`:** Full `cb-backend` rocm suite green + SC-4 `cargo tree` cubecl-free gate + `git diff --stat` showing `cb-compute`/`cb-core`/`cb-model` byte-unchanged
- **Max feedback latency:** ~90 seconds
