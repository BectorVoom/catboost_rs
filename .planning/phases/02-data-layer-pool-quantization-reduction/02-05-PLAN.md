## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| untrusted Arrow/Polars buffer → owned columns | malformed dtype / non-contiguous / NaN-in-categorical / length-mismatch must be a typed error, never a panic or blind index |
| class counts → class weights | div-by-zero on an empty class, or wrong summation order, breaks weight parity |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-13 | Tampering | Arrow/Polars buffer with mismatched dtype/length | mitigate | Validate `data_type()` + length at the boundary; return `CbError::Dtype`/length-mismatch; `indexing_slicing` denied |
| T-02-14 | Tampering | NaN smuggled into a categorical column | mitigate | Reject NaN in declared-categorical columns at ingestion → `CbError::NanInCategorical` |

| T-02-16 | Tampering | unaudited float sum in class-weight accumulation | mitigate | All counts/weights summed via `cb_core::reduction`; D-08 grep; oracle gate at 1e-5 |
