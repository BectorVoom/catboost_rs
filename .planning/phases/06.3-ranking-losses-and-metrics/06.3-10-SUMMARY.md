## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] catboost toolchain hardcodes bare `clang`/`clang++`**
- **Found during:** Task 1 (STEP 4 first build attempt)
- **Issue:** `build/toolchains/clang.toolchain` sets `CMAKE_C_COMPILER clang` / `CMAKE_CXX_COMPILER clang++` and re-exports `ENV{CC}`/`ENV{CXX}=clang/clang++`, OVERRIDING the `-DCMAKE_*_COMPILER=/tmp/clang18_prefix/.../clang-18` cache entries → CMake "is not a full path and was not found in the PATH".
- **Fix:** The driver creates sudo-free `clang` → `clang-18`, `clang++` → `clang++-18` (and a `clang-14` alias for the CUDA-host probe) symlinks inside the prefix bin, exports `CC`/`CXX` to them, and passes them as the cache compilers.
- **Files modified:** build_instrumented_trainer.sh
- **Verification:** CMake configure passed; build advanced to compilation.
- **Committed in:** dae8bff

**2. [Rule 1 - Bug] `awk -v` mangled the sink's `'\n'` into a real newline**
- **Found during:** Task 1 (STEP 4 second build attempt — `error_functions.cpp:21 expected expression`)
- **Issue:** Passing the C++ sink helper through `awk -v helper=...` let awk interpret the backslash in `std::fputc('\n', f)`, splitting the char literal across two lines and breaking all four patched TUs.
- **Fix:** Write the sink to a temp file and splice it via awk `getline` (no `-v`); replace `'\n'` with `std::fputc(10, f)` (ASCII LF, no backslash).
- **Files modified:** build_instrumented_trainer.sh
- **Verification:** the four TUs compiled past the sink; only the hook lines remained as the next error.
- **Committed in:** dae8bff

**3. [Rule 1 - Bug] perl `\"`-escaped JSON produced invalid C++ string literals**
- **Found during:** Task 1 (STEP 4 third build attempt — `:expected ')'` in three TUs)
- **Issue:** The perl `s///` replacement's `\"` escapes were consumed by perl, leaving bare `"` inside the injected `CbInstrumentLog("{"event"..." )` hooks → unterminated/invalid C++ literals.
- **Fix:** Switched every hook's JSON fragments to C++ raw string literals `R"J(...)J"`, which need no quote escaping; added `noise[docId]` to the StochasticRank event.
- **Files modified:** build_instrumented_trainer.sh
- **Verification:** clean build (RC=0), `_catboost.so` produced, smoke run emits valid JSONL.
- **Committed in:** dae8bff
